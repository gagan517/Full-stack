<!-- Theme Switch Script -->

<script>
Starts an inline JavaScript block.

const themeToggle = document.getElementById('theme-toggle');
Finds the checkbox by ID and stores it in the themeToggle variable.

const body = document.body;
Stores reference to the <body> element in body so we can change its classes.

`` (blank line)

themeToggle.addEventListener('change', () => {
Adds an event listener that fires when the checkbox changes state (checked/unchecked).

body.classList.toggle('dark', themeToggle.checked);
Adds the dark class to <body> if the checkbox is checked; removes it if unchecked. The second argument is a boolean (so toggle acts as set/unset).

body.classList.toggle('light', !themeToggle.checked);
Ensures the light class is present when the checkbox is NOT checked (keeps classes explicit).

});

/* Grid container for the layout */
Comment describing the .container block.

.container {
Start style for the grid wrapper used in HTML.

display: grid;
Turns .container into a CSS Grid layout — the core of the page layout.

grid-template-rows: 60px 1fr 40px;
Defines three row sizes: header (60px), content (flexible 1fr), footer (40px).

grid-template-columns: 200px 1fr;
Two columns: sidebar fixed at 200px and main content flexible.

grid-template-areas:
Begins naming the grid areas — makes placement semantic and readable.

"header header"
Row 1: header spans both columns.

"sidebar main"
Row 2: sidebar left, main content right.

"footer footer";
Row 3: footer spans both columns.

height: 100vh;
Makes the container fill the full viewport height (so footer sticks to bottom of screen).

}
End .container block.
